# Instruções para Substituição de Imagens - Portfólio

Para atualizar as fotos dos projetos no site, siga estas instruções para manter a qualidade e o layout perfeitos.

## Localização
As imagens devem ser salvas na pasta:
`public/projects/`

## Nomenclatura Padrão
Substitua os arquivos existentes mantendo os mesmos nomes, ou adicione novos e atualize o código:
1. `project-1.png` (Primeiro card)
2. `project-2.png` (Segundo card)
3. `project-3.png` (Terceiro card)

## Especificações Técnicas (Resolução)
O site utiliza uma proporção vertical (Retrato) de **4:5**.

**Resolução Recomendada:**
*   **Largura Mínima**: 1000px
*   **Altura Mínima**: 1250px
*   **Formato Ideal**: JPG (para fotos) ou PNG.
*   **Tamanho de Arquivo**: Tente manter abaixo de 500kb por imagem usando ferramentas como TinyPNG para garantir que o site carregue rápido.

**Exemplo de Tamanho Perfeito:**
`1200px (largura) x 1500px (altura)`

## Nota sobre Estilo
Como o site usa o conceito "Transcendental" com sobreposições estilo vidro (Glassmorphism), prefira fotos que tenham:
*   Boa iluminação natural
*   Contrastes suaves
*   Espaços "respirando" (evite fotos muito cheias de informação visual nas bordas)
